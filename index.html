Este dashboard lee la hoja "Nómina ARG" en vivo a través de una pequeña API de Google Apps Script (así nunca hay que subir ni descargar el Excel/Sheets manualmente).

  1. Abrí tu Google Sheets → menú Extensiones → Apps Script
  2. Pegá el código del archivo AppsScript_Code.gs que te compartí
  3. Implementar → Nueva implementación → tipo "Aplicación web", ejecutar como "Yo", acceso "Cualquier usuario de Farmatodo" (o "Cualquier usuario")
  4. Copiá la URL que termina en /exec
  5. Pegala en este archivo HTML, en la constante API_URL, al inicio del bloque <script>

Una vez configurado, este mensaje desaparece y el panel se actualiza solo cada vez que abras el archivo o toques "Actualizar ahora".
